Introducción:

Soy Alexis Gallegos, estudiante de Ingeniería en Sistemas

Y este es mi portafolio virtual para la asignatura de programación computacional IV, en cuya página plasmaré los contenidos y aprendizajes adquiridos durante el ciclo V del año 2022, de entre algunos de los temas que desarrollamos, vimos el uso de JavaScript como lenguaje de programación para frontend, conociendo sobre el DOM y las partes que debe contener una página web para que el cliente disfrute de la misma. También conocimos frameworks que se acoplan muy bien al fácil desarrollo de aplicaciones para el cliente como lo es: Vue.js y del lado del administrador, encontramos a Laravel que se maneja con lenguaje PHP.
